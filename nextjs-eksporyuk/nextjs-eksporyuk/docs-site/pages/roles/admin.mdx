## ğŸ› ï¸ Troubleshooting Common Issues

### Issue: User tidak bisa login
**Solusi:**
1. Check apakah user suspended
2. Verify email sudah verified
3. Reset password via admin panel
4. Check role assignment

### Issue: Transaksi tidak terbagi otomatis
**Solusi:**
1. Check **Pending Revenue** untuk transaksi tersebut
2. Verify commission settings di product/membership
3. Manual trigger revenue split via API (advanced)
4. Contact developer jika persistent

### Issue: Affiliate link tidak tracking
**Solusi:**
1. Check affiliate profile aktif
2. Verify link format correct
3. Check cookie settings (domain must match)
4. Review click logs di **Short Link** management
